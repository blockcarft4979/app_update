# Freewd 服务器介绍

Freewd 服务器是一个面向 **Minecraft 基岩版** 玩家的纯净生存公益服，旨在提供一个稳定、友好且专注于建设的游戏环境。其 **4.0 版本** 正式开启

## 核心特色
Freewd 服务器主打纯净、安心的生存体验，其主要特色包括：

*   **安心建造体验**：服务器禁用了苦力怕的爆炸破坏，有效保护玩家的建筑成果，让创意不受意外干扰。
*   **资源安全保护**：通过独家保护插件，保障玩家的个人物品安全，防止盗窃与误操作。
*   **友好社区氛围**：拥有活跃友善的玩家群体与管理团队，共同营造良好的社区环境。
*   **公益自由玩法**：服务器坚持纯净无收费模式，无强制捐赠，力求回归游戏最初的美好。

## 新玩家礼遇
为帮助新玩家快速起步，初次加入服务器可领取「启程礼包」，内含 **18颗钻石** 和 **25份面包**。
